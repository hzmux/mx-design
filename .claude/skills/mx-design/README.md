# mx-design

基于 [awesome-shadcn-ui](https://github.com/birobirobiro/awesome-shadcn-ui) 封装的 Claude Code 技能。

## 安装

```bash
curl -fsSL https://raw.githubusercontent.com/hzmux/mx-design/main/install.sh | bash
```

安装时会自动下载 awesome-shadcn-ui 组件数据。

## 使用

在 Claude Code 中输入：

```
/mx-design 帮我找个日历组件
/mx-design 我要做 AI Chat 应用
```

详细说明见 [INSTALL.md](./INSTALL.md)
